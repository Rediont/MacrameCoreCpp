Інструкція до динамічної бібліотеки MacrameCoreCpp

Назва бібліотеки:     MacrameCoreCpp.dll
Версія:                1.0
Платформа:             Windows 10 / 11 (x64)
Тип:                   Динамічна бібліотека (DLL)
Мова:                  C++
Призначення:           Ядро для Калькулятора Макраме (Macrame Calculator)

Бібліотека призначена для точного розрахунку параметрів макраме-проекту 
(панно, гобелен, килимок тощо). Вона приймає розміри панно та інші вхідні дані 
і повертає повний набір характеристик, які відображаються в програмі 
«Калькулятор Макраме (C++ DLL Core)».


ФУНКЦІЯ РОЗРАХУНКУ ПАРАМЕТРІВ МАКРАМЕ

Опис:
Основна функція бібліотеки виконує комплексний розрахунок усіх необхідних 
параметрів для плетіння макраме-панно заданого розміру. 
Результати відповідають тому, що ви бачите в консолі демо-програми.

Синтаксис (приклад для C# / P/Invoke):
[DllImport("MacrameCoreCpp.dll", CallingConvention = CallingConvention.StdCall)]
public static extern void CalculateMacramePanel(
    double widthCm,          // ширина панно в см
    double heightM,          // висота панно в метрах
    out int baseThreads,     // кількість ниток основи
    out double threadLength, // довжина однієї нитки
    out double totalCord,    // загальний метраж шнура
    out int knotsPerSection, // місткість вузлів у секції 15 см
    out double weavingTime,  // орієнтовний час плетіння (годин)
    out double materialCost, // орієнтовна собівартість матеріалів (грн)
    out char difficulty      // рівень складності: B / I / P
);

Параметри:
widthCm          – ширина панно в сантиметрах (наприклад 80)
heightM          – висота панно в метрах (наприклад 1.5)
baseThreads      – (out) кількість ниток для основи
threadLength     – (out) довжина однієї нитки для нарізки (метри)
totalCord        – (out) загальний метраж необхідного шнура (метри)
knotsPerSection  – (out) кількість вузлів у секції 15 см
weavingTime      – (out) орієнтовний час плетіння (години)
materialCost     – (out) орієнтовна собівартість матеріалів (гривні)
difficulty       – (out) рівень складності проекту (B=Новачок, I=Середній, P=Профі)

Повернене значення:
void (результати повертаються через out-параметри)

Приклад використання (результат, який ви бачите в програмі):

4. Місткість вузлів у секції 15см:     12 вузлів
5. Орієнтовний час плетіння:           8.3 годин
6. Орієнтовна собівартість матеріалів: 7521.60 грн
7. Рівень складності проекту:          P (Профі)

Контроль параметрів: не виконується (функція розрахована на коректні вхідні дані)
Особливі ситуації:   не визначені


ВИКОРИСТАННЯ БІБЛІОТЕКИ

1. У проєкті MacrameEngineDemo (C#)
   - DLL автоматично підключається через P/Invoke.
   - Після Rebuild Solution (x64) функція CalculateMacramePanel 
     викликається з Program.cs.

2. Неявне / явне зв’язування
   - Для C++ проєктів використовуйте __declspec(dllimport) та .lib файл.

3. Важливо:
   - Рішення має бути зібране тільки для платформи x64.
   - Файл MacrameCoreCpp.dll повинен лежати поруч з .exe або в x64\Debug\ / Release\.
